## Gaussian distribution of Random variable  
  
* 난수를 생성하는 함수를 이용하여 uniform distribution의 random variable을 gaussian 분포를 갖는 그래프를 생성한다.  
* 평균과 분산을 조정하여 Gaussian 분포가 달라지는것을 확인하고, 이때 생성된 그래프가 gaussian 분포인지 확인한다.
