# MFC 2048게임  

MFC로 2048게임을 구현한다.  
  
게임설명 : 2048은 10의 11제곱이다. 2를 총 1024번 더하여 2048을 만들게 된다. 한 줄에 같은 숫자가 2개 있다면 두 수는 합쳐져서 수의 2배의 값을 가지게 된다. 이렇게 해서 2048을 만들면 이기는 게임이다. 만약 칸을 모두 채우고 더 이상 움직일 수 없다면 게임에서 지게 된다.
